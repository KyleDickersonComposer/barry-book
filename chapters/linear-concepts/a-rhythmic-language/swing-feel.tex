\section{Swing Feel}
It's a common misconception that the \emph{jazz feel} is a 8th note triplet figure like this:

\bookExample{The Wrong Feel}{./excerpts/wrong-feel.ly}

For horn players, all we need to think about is tonguing the up-beats like this, but for guitar or piano players, we need to fake this effect by connecting the up-beats to the down-beats with a slur and to play the up-beat slightly louder. This process creates a natural dog-legged effect without being a silly 8th note triplet:
