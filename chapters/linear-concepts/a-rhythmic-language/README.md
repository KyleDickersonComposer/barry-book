# A Rhythmic Language

### Rough Outline
This will talk about how to get the swing feel down. Also, shows how it comes from this notion of an *unaccented appoggiatura* which is a deceptive and silly term. We will demonstrate and probably make video examples of why this kind of thing to show in a written and aural what the swing feel is like. Will also dispell myths about the swing feel being a *triplet feel* which is very corny. Also want to demonstrate how horn players seems to have better feel by default for a technical reason that they are tonguing the off 8ths to the beat with slurs etc. We will also cover the kinds of articulations and we want videos that interpret the same material in a classical style and in a jazz style just to demonstrate the difference audibly.
