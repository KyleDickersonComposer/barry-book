# Functional Harmony

### Rough Outline
This introduction to functional harmony will be quite different because it has no reference to cadences. This will only describe the functions in terms of tendency towards some area of the key. For example we will mention that there are two functions functions that have a tendency towards 1. 5->1 and 4->1. And one function that have tendency towards 5. 2->5. We will have to discuss the tritone as the generator of this tendency. And we might discuss some of its properties which distinguish it from other dissonance, like how it doesn't function the same way that other dissonances do, with their tendency to resolve down. We will mention that the tritone can resolve in many different way. I think we can discuss my concept of the dim. chord and tritone being functionality equivalent. And how you can think about how the pitches of a tritone, or a dim. chord resolving. And how the symmetrical properties of it allow for advanced chromatic possibilities that we will discuss in later chapters. If you look at those transforms for just a single note you will see that if you go down from any note of a dim. chord by whole step you will see that that note can be the tonic, third, or fifth of a chord based on that note. If you do this you will find third related chords that come from that relation etc. This is probably an easier way to look at the related chords, but I'm not totally sure need to work it out more.
