# Changes

### Rough Outline
This will discuss changes as the fundamental chords that go with the tune. And will make a distinction between the chords that the jazz tradition adds as a matter of convention to each tune. We will discuss Hank Jones and how he deals with changes, in terms of tune changes and variations on those, and in terms of solo changes and variations on those. We will talk about how the more simply you see the tune the more elaborate the changes you add can be. The vast majority of variations to the changes have to do mostly with finding various paths towards landing on the same functional locations via the tritone/dim. chord and its ambiguous properties. We will discuss how the melody and its cadence points determines what keys are tonicized and the various ways you can get from point A in a tune to point B.
