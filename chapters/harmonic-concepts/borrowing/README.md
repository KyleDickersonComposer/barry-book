# Borrowing

### Rough Outline
This is a tool for elaborating harmonies. It is analogous to the appoggiatura and can be thought to be almost identical with it. Except for that the appoggiatura is fundamentally a contrapuntal phenomena, with harmonic consequences. The borrowing concept is mostly about taking notes from the off chord, which is a dim. chord built one step above the root of a major or minor chord.
