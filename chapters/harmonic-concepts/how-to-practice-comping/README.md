# How to Practice Comping

### Rough Outline
Comping, like anything, is best learned one concept at a time. There are many ways to think about how to comp and to comp well. This section will show typical comping rhythms as applied to movements and how to connect the changes together with various things. Also, will rely on concept from before like syncopation and rhythmic games and so on to show how to compose some good comping. I do think that the like how to sections should really focus on composing a nice comping solo first before learning to improvise it right? We can also stick to models for a while before applying the models to the real tunes. Not really necessary to add that complexity.
